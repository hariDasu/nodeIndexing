Help : Modify text size

To modify text size, use the browser commands.

Safari

In the toolbar, select View > Zoom > Zoom In, Reset or Zoom Out. With the keyboard, press COMMAND + to zoom in, COMMAND - to zoom out or COMMAND 0 to reset.

Chrome

In the toolbar, select View > Zoom In, Actual Size or Zoom Out. With the keyboard, press COMMAND + (Mac) or CTRL + (Windows) to zoom in, COMMAND - (Mac) or CTRL - (Windows) to zoom out or COMMAND 0 (Mac) or CTRL 0 (Windows) to make actual size

Firefox

In the toolbar, select View > Zoom > Zoom In, Reset or Zoom Out. With the keyboard, press COMMAND + (Mac) or CTRL + (Windows) to zoom in, COMMAND - (Mac) or CTRL - (Windows) to zoom out or COMMAND 0 (Mac) or CTRL 0 (Windows) to make actual size

Internet Explorer

In the toolbar, select Options > Zoom > Zoom In, Custom or Zoom Out. With the keyboard, press CTRL + to zoom in, CTRL - to zoom out or CTRL 0 to reset.

About the School

ÉSIS offers an Information Studies program grounded in theory, supported by practical work experience, and integrally connected to the pulse and trends of the leading knowledge centres in the National Capital Region and beyond. With a wide variety of professors from a range of interdisciplinary backgrounds, a multi-cultural student body, and a strong commitment to student learning outcomes, uOttawa offers a first-class Information Studies education, all within a bilingual North American setting.
